## ✨ 三个模块，一个页面

| 🗂️ 待办事项 | 🤖 AI 聊天 | 🎵 音乐播放器 |
|:---:|:---:|:---:|
| 多人共享同步 | 微信风格对话 | 放松频道 + 搜索点歌 |
| JSON Blob 云端存储 | 自定义 AI 角色 | B 站 + 酷我音源 |
| URL hash 一键分享 | 解锁无限模式 | 全局快捷键控制 |

## 🚀 快速开始

```bash
# 直接用浏览器打开
start index.html

# 或者跑个本地服务器
npx serve .
```

## ⌨️ 快捷键

| 按键 | 功能 |
|:---:|:---:|

| `↑` `↓` | 音量 增大/减小 |
